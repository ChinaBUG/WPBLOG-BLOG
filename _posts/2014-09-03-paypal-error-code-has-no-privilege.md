今天客服反应，在使用支付宝支付时出现如下错误提示：

“错误代码HAS_NO_PRIVILEGE

说明：如果您不是因为本接口集成调试而看见该错误提醒，请联系本次请求来源网站，本错误属于网站集成接口的错误。”

怎么回事噢，检查了一下发现是支付宝没有自动续费，接口被停用了，没有权限噢。
